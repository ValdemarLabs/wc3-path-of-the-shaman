# PotS Library Template

Use this header for standalone PotS JASS libraries. Keep the section order unchanged.

```jass
/**
    MasterUI

    Author: Valdemar
    Version:

    Description:




    Credits:

    How to install:

    API:

**/
library MasterUI
    globals
        // Configuration
        // Usage-sensitive globals and constants go here.
    endglobals

    // Add short comments only around advanced sections.
endlibrary
```

## Checklist

- Replace `MasterUI` with the actual library name in both the header and declaration.
- Fill in `Version` when the task has a meaningful version value; otherwise leave it blank instead of inventing one.
- Keep advanced inline comments short.
- Add basic `globals` and configuration comments when those sections exist.
- Expand `How to install` and `API` when the library exposes setup steps or public functions.
